# Internal Materials

This directory contains PaceFlow development and audit materials that are not part
of the published Claude Code plugin runtime.

- `audit-tickets/`: historical audit tickets and handoff notes.
- `legacy/`: deprecated local maintenance scripts kept out of the repository
  root. Files in this directory may still be ignored by `.gitignore` if they are
  local-only tools.
- `skills/audit/`: internal PaceFlow audit skill, not published to users.
